# CosRay-App

CosRay 前端应用

## 快速开始

### IDE

本项目推荐使用 [Android Studio](https://developer.android.com/studio) 进行开发。请参考其[安装文档](https://developer.android.com/studio/install)进行安装。

### 本地开发

```bash
# 克隆代码库
git clone https://github.com/GRID-Cos-Ray-Detection-Reasearch-Group/CosRay-App.git
cd CosRay-App
# 打开 Android Studio 并导入项目
```
